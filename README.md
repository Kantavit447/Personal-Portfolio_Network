# Personal Portfolio Network

## 📋 เกี่ยวกับโปรเจกต์

**Personal Portfolio Network** คือเว็บไซต์พอร์ตโฟลิโอส่วนตัวที่พัฒนาขึ้นเพื่อแสดงผลงาน ความสามารถ และประสบการณ์ของฉันอย่างมืออาชีพ

โปรเจกต์นี้รวบรวมผลงานจากทั้งการเรียน (Assignment, Lab) และโปรเจกต์ใหญ่ (Final Project) เพื่อให้ผู้ที่สนใจ (เช่น นายจ้าง อาจารย์ หรือเพื่อนร่วมงาน) สามารถดูผลงานของฉันได้อย่างครบถ้วนและทันสมัย

## ✨ คุณสมบัติหลัก

- ออกแบบ Responsive รองรับทุกขนาดหน้าจอ (Desktop / Tablet / Mobile)
- แสดงผลงานแบบจัดหมวดหมู่ชัดเจน (Assignment, Lab, Final Project)
- มีส่วนแสดงข้อมูลส่วนตัว ประวัติการศึกษา และทักษะ
- อินเทอร์เฟซที่สวยงาม ทันสมัย และใช้งานง่าย
- โหลดเร็วและเป็นมิตรกับ SEO

## 🛠️ เทคโนโลยีที่ใช้

- **Frontend**: HTML, CSS, JavaScript (หรือ React/Vue ถ้าใช้)
- **Styling**: Tailwind CSS / Bootstrap / Vanilla CSS
- **อื่นๆ**: (เพิ่มตามที่คุณใช้จริง เช่น GSAP สำหรับ animation, Three.js ฯลฯ)
